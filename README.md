# 🦖 ChompPDF 🦈

> 把 PDF 變小的小工具。檔案不離開你的電腦，安心用。

## 馬上用

**線上版**（點開就能用）：[https://ahlung.github.io/chomp-pdf](https://ahlung.github.io/chomp-pdf)

或者下載整個資料夾，雙擊裡面的 `index.html`，在瀏覽器打開就好 —— 不用安裝、不用帳號、不用網路。

## 適合什麼情況

- Gmail / LINE 說「檔案太大」寄不出去
- 履歷 / 提案 / 報告要上傳系統但卡大小限制
- 手機存一堆 PDF 想瘦身
- 掃描文件、相片 PDF 動輒幾十 MB 想壓小

## 怎麼壓

1. 把 PDF 拖進網頁（或點一下選檔）
2. 工具會自動幫你算出「建議縮到多少 MB」
3. 按一下**壓縮**，等一下就好
4. 下載結果

想自己設定大小、或整頁變圖片讓檔案更小，也可以調整。

## 為什麼可以「離線用」

整個工具就是一個 HTML 檔，所有壓縮功能（JPEG / PNG 引擎）都包在裡面。打開後完全在你瀏覽器跑，**檔案不會上傳到任何地方**。

## 更新記錄

### v1.4.8（2026-04-30）
- **頁面標題完全不糊了** — 文字遮色片直接 skip 不縮放（之前嘗試「縮+二值化」對小字仍會糊）。檔案大小影響極小（全部遮色片佔不到 3%）

### v1.4.7（2026-04-30）
- **真的修好頁面標題模糊** — v1.4.6 的偵測抽樣太稀，標題字密集區被空白行稀釋判定失敗。這版抽樣密度提高 5×、判定改成「雙峰強或邊緣多」二擇一，命中後輕度縮放 + 二值化 snap 字邊回 binary，銳利不糊還能省檔。

### v1.4.6（2026-04-30）
- **修正頁面標題仍被壓糊的真兇** — 標題文字其實是燒在「透明遮色片」裡，而瘦身遮色片那條路徑沒做文字偵測 → 銳利字邊被無差別縮放鋸齒化。現在加了文字遮色片偵測，命中就保持原解析度。

### v1.4.5（2026-04-30）
- 大幅減少壓縮中「網頁未回應」對話框 — 每張圖編碼後讓瀏覽器喘息一下
- 文字圖偵測門檻放寬，寬扁的標題 banner 也能進（原本被尺寸 guard 擋掉導致頁面標題仍被壓糊）

### v1.4.4（2026-04-30）
- **自動最佳化模式速度大幅提升**：每張圖只解碼一次給兩個品質版本共用（之前各解碼一次）— 大檔可省 30-50% 探測時間
- 遮色片瘦身內聯到預處理階段，大檔節省一次完整 PDF 序列化（~30-60 秒）

### v1.4.3（2026-04-30）
- **新增文字圖偵測**：整頁信息圖、表格、文字海報這類「畫成一張圖呈現的內容」會自動偵測並保留高畫質，文字不再被壓糊
- 大檔（>200MB）模式：跳過元件 / 色彩描述去重，省幾秒卡頓

### v1.4.2（2026-04-30）
